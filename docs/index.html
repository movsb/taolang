The Tao Programming Language

Tao 是由 tao 开发的一门类似 JavaScript 的脚本语言。 她是作者在学习编译原理期间，在下班后熬了 7 个晚上的时间打造出来的。希望你能喜欢。

她目前至少拥有以下特性：

  • 动态类型
  • 数组、对象
  • 函数作为第一类值
  • Lambda 表达式
  • 函数式编程
  • 词法作用域、闭包
